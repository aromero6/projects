SAM Registrations API

sam.gov is the System for Award Management. It is the site on which all vendors, grantees, receivers of federal financial assistance, and other entities doing business with the government must register prior to receiving an award.

The Registrations API currently provides the publicly available data for a specific entity along with status and progress information for that registration.

  • get/v1/registrations/{registrationId}

    • Find a registration by its DUN+4 number

    Implementation Notes

    If the entity only has a 9-digit DUNS number, pad the end to include 4 zeros
